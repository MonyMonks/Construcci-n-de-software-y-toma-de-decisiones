HTML

HTML is not a programming language; is a markup language that defines the structure of your content. HTML consists of a series of elements that you will use to enclose different parts of content so that they look or behave in a certain way.






Basic elements

  • body: This element is shown inside the main browser window (body).
  • head: This contains information about the page (h1).
  • title: These elements are either shown in the top of the browser, above where you usually type in the URL of the page you want to visit, or on the tab for that page (title).
  • paragraph: To create a paragraph (p) .
  • bold: Make characters appear bold (b).
  • breaking lines: Add a line break inside the middle of a paragraph (br).

Lists

  • Order Lists: ol and each item in the list is placed between an opening li.
  • Unordered Lists: The unordered list is created with the ul element.
  • Nested Lists: Add a second list inside an li element.
  • Images: We use an img element to add an image, using two other attributes that specify its size: height (this specifies the height of the image in pixels) and width (this specifies the width of the image in pixels).

JavaScript

It is an interpreted programming language, a dialect of the ECMAScript standard. It is defined as object-oriented, prototype-based, imperative, weakly typed and dynamic.

  • Variables: numbers (13), arithmetic (100 + 4 * 11 = 144), strings, concatenate("con" + "cat" + "e" + "nate"), unary operators (console.log(typeof 4.5); // → number; console.log(typeof "x");// → string), boolean values, logical operators, Short-circuiting of logical operators (console.log(null || "user"); // → user; console.log("Agnes" || "user"); // → Agnes ), functions.

Let, const and var:

  • var: has function scope and can be re-declared and updated.
  • let: allows reassignments.
